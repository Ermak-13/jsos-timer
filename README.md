# JSOS Timer

JSOS Timer is module of [JSOS](https://github.com/Ermak-13/jsos) project
and it is simple timer :)

## Gulp Commands
* `gulp default` - gulp dev
* `gulp dev` - watch, build module
* `gulp server` - run webserver on **localhost:9991**
